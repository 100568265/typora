## 1.2 空间复杂度



算法原地工作：表示算法所需内存空间为常量

**变量需要一些内存空间的开销：**

```cpp
void test(int n){
  int flag[n][n];
  int other[n];
  int i;
  //...
}
```



**函数递归调用带来的内存开销：**

```cpp
void loveYou(int n){
  int a,b,c;
  //...
  if(n>1){
    loveYou(n-1);
  }
  printf("I love you%d\n",n);
}
```

函数调用栈，最后当n=1时才结束

空间复杂度=递归调用的深度
